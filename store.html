🔒 安全审计 Security Audit
自动记录所有 Agent 工具调用日志，包含时间、工具名称、决策结果（允许/拒绝/需确认）及原因。数据持久化到 ~/.openharness/audit/tool_calls.jsonl，支持事后审计与异常行为回溯。 Logs every agent tool call to ~/.openharness/audit/tool_calls.jsonl with timestamp, tool name, decision (allowed/denied/pending) and reason. Enables post-hoc audit and anomaly tracing.
✅ 已集成到系统Built-in v0.6.78
⛔ 熔断器 Circuit Breaker
当 API 连续失败 5 次后自动跳闸，后续请求毫秒级拒绝而非等待超时。30 秒后台自动探测恢复，支持 Anthropic 与 OpenAI 兼容协议（DeepSeek、Kimi 等第三方 API 均覆盖）。 Trips after 5 consecutive API failures — subsequent requests fail instantly instead of timing out. Auto-probes recovery every 30s. Supports both Anthropic and OpenAI-compatible APIs (DeepSeek, Kimi, etc.).
✅ 已集成到系统Built-in v0.6.78
🔄 工具错误重试 Tool Error Retry
当工具调用因临时网络错误（DNS 解析失败、连接重置等）失败时自动重试一次，避免偶发网络抖动导致整个任务中断。仅对可重试错误触发，不重复执行非幂等操作。 Automatically retries once on transient network errors (DNS failures, connection resets, etc.), preventing intermittent network issues from aborting the entire task. Only triggers on retryable errors.
✅ 已集成到系统Built-in v0.6.78
