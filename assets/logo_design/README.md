# One 品牌 Logo 设计规范

> 程序化生成 | 非对称漩涡涡流 | 可无限缩放矢量源文件

---

## 设计理念

以 **“进化涡流”** 为核心概念，形态源于阿基米德螺旋，但通过三重非对称扰动打破太极式的完美平衡：
- 半径增长带正弦波动（打破圆规整性）
- 角度偏移带余弦波动（消除对称轴）
- 颜色渐变横跨冷暖（映射 One 的双螺旋定位：理性/算力 + 温度/伴侯）

中心 **奇点** 代表智能原生的“源头”——所有 Token 与能量向内汇聚。

## 三种官方版本

### 1. `one_logo_full.svg` — 数字主视觉版
**使用场景**：官网 Hero 区、App 启动页、品牌宣传图、社交媒体头图

- 流体渐变：电子青 → 深紫 → 洋红 → 暖金
- 中心奇点带柔和光晕滤镜
- 稀疏星尘粒子（约 35 颗，不影响主体识别）
- 圆点叠加自然形成密度渐变

### 2. `one_logo_mark.svg` — 单色符号版
**使用场景**：favicon（32×32）、GitHub Org Icon、小程序图标、产品内嵌小标识

- 纯色：`#7C3AED` （深紫，One 品牌主色）
- 无渐变、无光晕、无星尘
- 透明度从外圈向中心逐渐加深，保留立体深度
- 在任意尺寸下保持边缘清晰

### 3. `one_logo_flat.svg` — 打印实体版
**使用场景**：名片、T 恤、海报印刷、商务合同、线下背板

- 四段平涂色块：电子青 / 深紫 / 洋红 / 暖金
- 无渐变、无光晕、无透明效果
- 粗线条 stroke，圆角 linecap
- 实体物料印刷不失真

## 品牌色盘

| 色名 | HEX | 使用建议 |
|------|-----|-----------|
| 电子青 | `#00E5FF` | 外圈能量、冷启动元素 |
| 深紫（主色） | `#7C3AED` | 品牌标识、单色版、主按钮 |
| 洋红 | `#EC4899` | 过渡层、高亮提示 |
| 暖金 | `#FBBF24` | 奇点、重要数据、成功状态 |
| 背景白 | `#FFFFFF` | 通用背景 |
| 轻灰底 | `#F3F4F6` | 卡片/区块底色 |

## 字体搭配建议

- **英文标准字**：`Inter` 或 `Space Grotesk`（几何无衬线，结构感强，与流动的螺旋形成强烈对比）
- **中文标准字**：`阿里巴巴智能体 B1` 或 `腾讯体` 无字重版
- **排版原则**：Logo 与文字水平居中对齐，不强制比例，保持适当静谧距离

---

## 最小使用尺寸

| 版本 | 最小宽度 | 备注 |
|------|---------|------|
| full | 120 px | 低于此尺寸时光晕细节丢失 |
| mark | 16 px | favicon 场景可用，建议 32 px 起 |
| flat | 40 mm | 印刷场景建议，过小时色块融合 |

## 后续优化方向

1. **SVG → PNG 批量转换**：使用 `rsvg-convert` 或 `cairosvg` 生成 32/64/128/512/1024 多尺寸 PNG 套件
2. **动态版本**：基于 full 版制作 CSS/SVG 动画（螺旋缓转 + 奇点脉冲）
3. **开屏动画**：App 启动时螺旋从外向内收敛至奇点，1.2s 完成
4. **暗色模式**：定义纯黑背景 `#0A0A0A` 下的反色使用规范
